## tmux

Rebind `prefix` to <kbd>Ctrl</kbd><kbd>A</kbd>

**split window** <kbd>Prefix</kbd> + <kbd>\\</kbd><kbd>-</kbd>

**resize pane** <kbd>Shift</kbd> + <kbd>←</kbd><kbd>↑</kbd><kbd>→</kbd><kbd>↓</kbd>

**vi-style** yank/paste
